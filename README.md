# Generative UI Sandbox Lab

A local-first lab for policy-checked adaptive UI patches and constrained generated-widget runtime contracts.

It uses deterministic synthetic fixtures to demonstrate canonical stems, reversible user-specific patches, trusted declarative rendering, generated-widget manifest validation, sandboxing, privacy splits, and deterministic refresh.

## Development

```bash
npm install
npm run dev
```

Run `npm test`, `npm run lint`, and `npm run build` to verify the project.
